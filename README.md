# Max-Store-data

1) Project Overview
  (a) This project presents an interactive Sales Dashboard built using Power BI to analyze retail data from Max Store. The dashboard provides insights into sales            performance, customer behavior, and product trends to support data-driven decision-making

2) Dataset Description
   (a) Product details (Category, Size , SKU, Amount)
   (b) Sales data (Revenue, Quantity,Channel)
   (c) Customer information (Cust_id, Age, Gender)
   (d) Location Data (Country, ship-city, ship-state)
   (e) Orders Details (Order_id, Order_date)
