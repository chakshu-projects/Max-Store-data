# Max-Store-data


### 1) Project Overview  
       This project presents an interactive Sales Dashboard built using Power BI to analyze retail data from Max Store. The dashboard provides insights into sales            performance, customer behavior, and product trends to support data-driven decision-making

### 2) Dataset Description  
      (a) Product details (Category, Size , SKU, Amount)  
      (b) Sales data (Revenue, Quantity,Channel)  
      (c) Customer information (Cust_id, Age, Gender)  
      (d) Location Data (Country, ship-city, ship-state)  
      (e) Orders Details (Order_id, Order_date)    
### 3) Dashboard Features  
       (a) Sales overview(Total Sales, Quantity, MTD Sales)  
       (b) Region-wise performance analysis  
       (c) Category and Channels insights  
       (d) Interactive filters and slicers for dynamic exploration
       
### 4) Tools & Technologies used  
       (a) Power Bi  
       (b) Data Cleaning & Transformation (Power Query)  
       (c) Data Modeling    
       (d) DAX(Data Analysis Expressions)
### 5) Key Insights  
       (a) The State Maharashtra has highest revenue  
       (b) The Set is the top-selling product category  
       (c) March is considered as the peak month for the store  
       (d) The adults age-group has the highest contribution in sales
